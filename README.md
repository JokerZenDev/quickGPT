# QuickGPT: ChatGPT at Your Fingertips

## Overview
QuickGPT is a sleek, macOS menu bar application designed to bring the power of OpenAI's ChatGPT directly to your desktop, offering instant AI-powered conversations without the need to switch contexts or navigate through browser tabs. Built with efficiency and simplicity in mind, QuickGPT integrates seamlessly into your workflow, allowing for real-time interactions with ChatGPT through a minimalist interface positioned right within your menu bar.

## Features
Instant Access: Communicate with ChatGPT directly from your menu bar, with just a click or a global shortcut.
Simple Setup: Easy integration with your OpenAI API key through a user-friendly settings panel.
Efficient Workflow Integration: Designed for professionals, developers, writers, and anyone looking to boost their productivity by integrating AI assistance into their daily tasks.
Customizable Shortcuts: Configure global keyboard shortcuts to activate the chat window, enabling swift access to AI without disrupting your workflow.
Secure and Private: Your conversations and API key are stored securely, ensuring privacy and confidentiality.
Open Source: Dive into the code, contribute, and customize QuickGPT to fit your needs. Community contributions are welcome!

## Getting Started
1. Clone the Repository:
```bash
git clone https://github.com/JokerZenDev/QuickGPT.git
```
2. Install Dependencies:
Navigate to the project directory and install the required dependencies.

3. Set Up Your OpenAI API Key:
Open the application, navigate to the settings, and enter your OpenAI API key.

4. Enjoy Conversing with AI:
Access QuickGPT from your menu bar or using a customizable global shortcut.
